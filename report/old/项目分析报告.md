# UnusualChineseChess 项目分析报告

## 项目概述

**UnusualChineseChess** 是一个基于 Java 开发的中国象棋游戏，其核心特点是支持高度可定制的游戏规则系统。项目采用模块化设计，允许玩家通过配置不同的规则组合来创造独特的游戏体验。

### 基本信息
- **项目名称**: UnusualChineseChess
- **版本**: 2.1.0
- **技术栈**: Java 11 + Swing + Gson
- **构建工具**: Maven
- **许可证**: GPL-3.0

## 架构分析

### 1. 整体架构

项目采用经典的 MVC 架构模式：

- **Model层**: 游戏数据模型（[`Piece`](src/main/java/io/github/samera2022/chinese_chess/model/Piece.java), [`Board`](src/main/java/io/github/samera2022/chinese_chess/engine/Board.java), [`Move`](src/main/java/io/github/samera2022/chinese_chess/model/Move.java)）
- **View层**: Swing UI组件（[`ChineseChessFrame`](src/main/java/io/github/samera2022/chinese_chess/ui/ChineseChessFrame.java), [`BoardPanel`](src/main/java/io/github/samera2022/chinese_chess/ui/BoardPanel.java)）
- **Controller层**: 游戏引擎（[`GameEngine`](src/main/java/io/github/samera2022/chinese_chess/engine/GameEngine.java)）和规则系统

### 2. 核心模块

#### 2.1 游戏引擎模块 ([`GameEngine`](src/main/java/io/github/samera2022/chinese_chess/engine/GameEngine.java))
- 管理游戏状态和流程
- 处理走子逻辑和胜负判定
- 支持撤销、重开、回放功能
- 提供游戏状态序列化/反序列化

#### 2.2 规则系统模块
- **规则注册表** ([`RuleRegistry`](src/main/java/io/github/samera2022/chinese_chess/rules/RuleRegistry.java)): 定义所有可配置规则
- **规则配置** ([`GameRulesConfig`](src/main/java/io/github/samera2022/chinese_chess/rules/GameRulesConfig.java)): 管理规则状态
- **走子验证器** ([`MoveValidator`](src/main/java/io/github/samera2022/chinese_chess/rules/MoveValidator.java)): 根据规则验证走子合法性
- **将军检测器** ([`CheckDetector`](src/main/java/io/github/samera2022/chinese_chess/rules/CheckDetector.java)): 检测将军状态

#### 2.3 网络模块
- **网络会话** ([`NetworkSession`](src/main/java/io/github/samera2022/chinese_chess/net/NetworkSession.java)): 处理网络通信
- **网络控制器** ([`NetModeController`](src/main/java/io/github/samera2022/chinese_chess/net/NetModeController.java)): 管理联机游戏

#### 2.4 UI模块
- **主窗口** ([`ChineseChessFrame`](src/main/java/io/github/samera2022/chinese_chess/ui/ChineseChessFrame.java)): 整合所有UI组件
- **棋盘面板** ([`BoardPanel`](src/main/java/io/github/samera2022/chinese_chess/ui/BoardPanel.java)): 渲染棋盘和棋子
- **规则设置面板** ([`RuleSettingsPanel`](src/main/java/io/github/samera2022/chinese_chess/ui/RuleSettingsPanel.java)): 动态生成规则配置界面
- **信息侧面板** ([`InfoSidePanel`](src/main/java/io/github/samera2022/chinese_chess/ui/InfoSidePanel.java)): 显示网络状态和游戏信息

## 功能特性分析

### 1. 规则定制系统

项目最核心的特性是**可扩展的规则系统**，目前支持 28 种不同的规则配置：

#### 核心设置 (outside)
- 允许悔棋 (ALLOW_UNDO)
- 显示提示 (SHOW_HINTS)
- 允许强制移动 (ALLOW_FORCE_MOVE)

#### 延申玩法 (extended)
- 允许飞将 (ALLOW_FLYING_GENERAL)
- 取消对将 (DISABLE_FACING_GENERALS)
- 允许出仕 (ADVISOR_CAN_LEAVE)
- 国际化王/仕 (INTERNATIONAL_KING/ADVISOR)
- 兵卒相关规则 (PAWN_PROMOTION, PAWN_CAN_RETREAT 等)
- 取消卡子/马脚/象眼 (UNBLOCK_PIECE/HORSE_LEG/ELEPHANT_EYE)

#### 特殊玩法 (special)
- 左右连通 (LEFT_RIGHT_CONNECTED)
- 死战方休 (DEATH_MATCH_UNTIL_VICTORY)
- 棋子堆叠系统 (ALLOW_PIECE_STACKING, MAX_STACKING_COUNT 等)

### 2. 网络对战功能

- **局域网对战**: 支持主机/客户端模式
- **实时同步**: 走子、规则变更、游戏状态同步
- **强制走子协商**: 通过请求-确认机制实现特殊走子

### 3. 游戏状态管理

- **历史记录**: 完整的走子和规则变更历史
- **回放功能**: 支持任意步数回放
- **导入导出**: JSON格式的游戏状态保存和加载

### 4. 强制走子机制

独特的强制走子功能允许玩家：
- 申请执行非传统走法
- 需要对方确认才能生效
- 支持重试和超时机制

## 技术亮点

### 1. 模块化设计
- 规则系统与游戏逻辑完全解耦
- 新增规则只需在 [`RuleRegistry`](src/main/java/io/github/samera2022/chinese_chess/rules/RuleRegistry.java) 注册
- UI自动根据规则定义生成配置界面

### 2. 扩展性
- 开发者友好的规则添加机制
- 详细的开发文档 ([`如何创建新玩法(2.X.X).md`](docs/如何创建新玩法(2.X.X).md))
- 依赖和冲突关系自动管理

### 3. 网络同步
- 完整的游戏状态同步机制
- 规则变更的实时传播
- 强制走子的协商协议

### 4. 错误处理
- 完善的异常捕获和日志记录
- 网络连接失败的重连机制
- 游戏状态不一致的恢复策略
